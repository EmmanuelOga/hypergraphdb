Introduction

Thank you for downloading the HyperGraphDB distribution, release 1.0. HyperGraphDB is an innovative database management system geared towards the complexities and open-endedness of modern, data intensive information systems, in particular the semantic web, artificial intelligence, networks research and object-orientation.

HyperGraphDB is an embedded database, based on the Java platform and the robust and high-performance BerkeleyDB storage system.

System Requirements and Installation

  • Java 5 or later is required to compile and/or run HyperGraphDB.
  • A fairly recent version of Linux 32bit/AMD64 or Windows 32bit/AMD64 is required for prebundled BerkeleyDB libraries to run, OR
  • Any platform that will run Java 5 and BerkeleyDB.

HyperGraphDB is open-sourced software licensed under the GNU Lesser General Public License. This essentially means that you are allowed to use it for any purpose you see fit, free of charge. Unlike GPL, the LGPL doesn't require you to make the source code of your software available for free.

Documentation

The following documentation is currently available:

  • Tutorial - walks you over most of what you need to develop real-world applications with HyperGraphDB.
  • HyperGraphDB Application - APIs and application integrating HyperGraphDB with other software such WordNet and a Prolog language interpreter as well as implementions of industry standards such topics maps and OWL (upcoming).
  • An API reference in the form of Javadocs is available online and it is also included within this distribution for offline use:
    • Online HyperGraphDB Javadocs
    • Offline HyperGraphDB JAvadocs
    Since we update the online documentation with each new release, the offline version will be more accurate for the release contained within this distribution.
